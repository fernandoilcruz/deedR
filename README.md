# An R package for DEEDados API database

## Introduction

deedR is an R package for downloading socioeconomic data for the state of Rio Grande do Sul using DEEDados API.
 The DEEDados public API was developed by Josu√© Klafke Sperb from the Department of Economics and Statistics - Planning, Governance and Management Office at State Government of Rio Grande do Sul (DEE/SPGG-RS). The API documentation can be accessed [here](https://dados.dee.rs.gov.br/api/).
